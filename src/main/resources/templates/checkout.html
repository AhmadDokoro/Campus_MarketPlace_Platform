Integration notes:
  • Your server should accept a POST with order items, amount and customer info then call Tayyib's API to create a payment session.
  • Server returns a payment URL or payment token to the client; client then redirects user to the gateway or opens the checkout widget.
  • After payment, Tayyib will call your callback/notify URL — record payment verification server-side and update the order status.
  • Do not embed API secrets in the frontend. Store them on your server (environment variables) and use server-to-server calls.
